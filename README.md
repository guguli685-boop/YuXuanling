# 影迹 (Trace of Image) v2.2
  影迹 是一款基于 TMDB 全球数据库打造的极简主义电影/美剧检索工具。它将强大的数据检索能力与现代化的液态视觉设计相结合，为您提供丝滑的观影探索体验。


✨ 核心功能

🔍 深度检索系统

海量数据：集成 TMDB 全球数据库，支持搜索电影、电视剧、美剧及动漫信息。

双布局切换：支持“竖屏海报”与“横屏剧照”两种视图，满足不同审美偏好。

热门动态：首页自动同步全球当日流行影视趋势。

🛠️ 独家特色工具

多地区别名配置：

支持中国大陆、台湾、香港、美国、日韩、英法德等多地译名配置。

排版优化：选项采用垂直排列，确保长词不重叠，显示清晰完整。

幸运随机看：内置随机算法，从高分榜单中一键抽取作品，告别“选片困难症”。

译名一键复制：详情页点击任意地区别名即可快速复制到剪贴板。

👤 个人化管理

收藏夹：支持“想看”清单管理，采用本地持久化存储。

检索历史：自动记录最近 10 条搜索历史，支持一键清空或快速二次检索。

自定义 API Key：支持输入个人的 TMDB API 密钥，确保数据访问的独立与安全。

🎨 极美视觉体验
液态动态背景：随鼠标操作产生微位移与缩放的流光渐变背景。

iro.js 调色盘：内置实时拾色器，支持全站主题色自定义应用。

深/浅色模式：完美适配不同光线环境，支持一键切换并自动保存设置。

响应式设计：完整适配 PC 及移动端，手机端拥有独立的抽屉式侧边栏交互。

🚀 快速开始
下载本项目的 index.html 文件。

在浏览器中打开该文件。

点击左下角的 盾牌图标 (API 配置)，输入您的 TMDB API Key 即可开始使用。

🛠️ 技术栈
HTML5 / CSS3：采用原生 CSS 变量实现主题切换，结合 Flexbox/Grid 响应式布局。

JavaScript (Vanilla JS)：纯原生 JS 逻辑，无臃肿框架，秒速加载。

iro.js：强大的前端交互拾色器插件。

FontAwesome：高清矢量图标库。
